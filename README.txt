Инструкция для ботанов
-----------------------------------
Если хочешь почитать исходный код, и посмотреть как и что работает, запускай .ipynb файл

Если хочешь просто прочитать работу, ничего не меняя, то открой .pdf файл (там 60 страниц так что УДАЧИ!)

-------------------------------------
Инструкция для подсосов
-------------------------------------
В файле output.xlsx лежат все базовые таблицы, которые я кидал в отчет для большинства систем, их менять не нужно

Для периодических систем я брал таблицы из ФОФО и ФОДО .xlsx , их менять не нужно

в папке output_images вы найдете графики, которые уже я запихнул в отчет для 2-го, 3-го и 4-го задания, их менять не нужно

в папке period_images лежат графики для 5 задания, их менять не нужно

В папке Варианты для задачи 5.б по вариантам лежат графики, которые нужно заменить в отчете для РИСУНКА №9 (в зависимости от вашего варианта)

В папке Варианты для задачи 6.г по вариантам лежат графики, которые нужно заменить в отчете для РИСУНКА №12 (в зависимости от вашего варианта)

В папке Варианты для задачи 7 по вариантам лежит вся информация для различных сложных систем (в зависимости от вашего варианта)

В этой же папке вы найдете xlsx таблицу со своими значениями для зависимости выходных характеристик, просто копируете таблицу и вставляете в форме 

ФОТОГРАФИИ, так лучше всего вставляется в отчет (то что в моем отчете, соответственно, удаляете)

p.s. Если у вас нет построенной бета-функции для вашего варианта, а вместо него файл txt error, то ничего страшного, просто для вашей системы не выполняется
условие сходимости. Я спросил у преподов, такое может быть, поэтому ничего страшного не произошло, просто не пишите об этом в отчете.

Важный факт: В последнем задании вы можете увидеть на графиках резкий скачок при Value 6 у некоторых параметров. Это моя лажа, но мне лень ее менять.
